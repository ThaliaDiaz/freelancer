# Freelancer

Creación de un sitio web interactivo imitando la página de Bootstrap.

## Objetivo

* El objetivo de este trabajo es practicar los temas de posicionamiento, tipografía, íconos elementos y pseudolementes.

* Practicar el uso de hover y active.

## Herramientas

* Html 5
* Css 3
* Font Awesome
